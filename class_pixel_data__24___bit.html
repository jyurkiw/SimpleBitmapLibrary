char * PixelData_24_Bit::pack_pixel ( ) [virtual]

Implimentation of required pack_pixel() function.

Packs the pixel data into a char array and returns it. 24-bit pixel data is contained in 3 bytes so the return array is 3 chars long (blue, green, red).
